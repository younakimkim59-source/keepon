# 🎁 KEEPON - 가족 선물 지갑

> Keep your gifts on time.

가족이 함께 쓰는 선물 지갑입니다. 흩어진 기프트콘을 한눈에 정리하고, 유효기간이 임박하면 잊지 않게 알려줍니다.

## 🎯 해결하는 문제

### 기존 문제점
- 📱 여러 플랫폼에 흩어진 기프트콘 관리의 어려움
- ⏰ 유효기간 놓치며 낭비되는 기프트콘
- 👨‍👩‍👧 가족 간 정보 공유의 어려움
- 💸 받은 선물을 잊고 사용하지 못하는 경우

### Keepon의 해결
✅ 여러 플랫폼을 한 곳에서 통합 관리  
✅ 유효기간 임박 시 자동 알림  
✅ 가족 단위 공동 지갑으로 쉬운 공유  
✅ 한눈에 보는 통계와 정렬  

## ✨ 주요 기능

### 1. **기프트콘 CRUD**
- ✅ **추가**: 플랫폼, 브랜드, 상품명, 금액, 유효기간, 메모 입력
- 📋 **조회**: 전체 목록 보기 및 통계
- ✏️ **수정**: 기프트콘 정보 변경
- 🗑️ **삭제**: 사용 완료 후 제거

### 1.5. **여러 플랫폼 통합 관리**
- 🎁 카카오톡 선물하기, 💰 네이버페이, 💳 토스
- 🍔 배달의민족, 🚗 요기요, 🛒 컬리, 🏪 SSG
- 📦 쿠팡, G마켓, 옥션, 11번가 등
- 여러 플랫폼에서 받은 기프트콘을 한 곳에서 관리

### 2. **가족 공유 기능**
- 👨‍👩‍👧 가족 코드(PIN) 기반 공동 지갑
- 동일한 코드를 입력하면 같은 지갑에 접근
- 로컬스토리지에 안전하게 저장

### 3. **유효기간 알림**
- 🟡 D-3: 노란색 표시
- 🔴 D-1: 빨간색 표시 및 브라우저 알림
- ⚫ 만료: 회색 표시

### 4. **검색 및 필터**
- 🔍 실시간 검색 (플랫폼명, 브랜드명, 상품명)
- 📂 필터링 (전체, 사용가능, 곧만료, 만료됨)
- 🏢 플랫폼별 필터 (원하는 플랫폼만 보기)

### 5. **통계 대시보드**
- 📦 전체 기프트콘 개수
- ⏰ 곧 만료되는 기프트콘 수 (D-3)
- 💰 총 금액
- 🏢 사용 중인 플랫폼 개수

## 🚀 사용 방법

### 페이지 구조 (총 3개)
```
1️⃣ login.html (로그인)
    ↓
2️⃣ dashboard.html (메인 - 지갑 관리)
    ↔
3️⃣ about.html (서비스 소개)
```

### 1. 시작하기
1. **`login.html`** 파일을 브라우저에서 엽니다
2. 가족 코드를 생성하거나 기존 코드를 입력합니다
3. **dashboard.html**로 이동하여 기프트콘을 추가하고 관리합니다
4. **about.html**에서 서비스 소개를 확인할 수 있습니다

### 2. 가족과 공유하기
1. 화면에 표시된 "가족 코드"를 가족에게 공유합니다
2. 가족이 같은 코드로 로그인하면 같은 지갑을 볼 수 있습니다

### 3. 알림 권한 설정
- 브라우저에서 알림 권한을 허용하면 유효기간 임박 시 알림을 받을 수 있습니다
- Chrome: 자동으로 알림 권한 요청
- Edge: 자동으로 알림 권한 요청
